# pet-boarding-website

<h3>Features</h3>
<ol>
  <li>Home page with sorting, filtering and pagination features.</li>
  <li>User can login to book session with a pet board house.</li>
  <li>User can add or edit or remove their datails.</li>
  <li>User can add or edit or remove their booking details.</li>
  <li>Admin can handle overall data of petboards, users, and pets.</li>
  <li>Only admin can approve booking session of user.</li>
  <li>For security perspective users passwords will be stored in hash format using bcypt package.</li>
  <li>For Admin Login use https://pet-boarding-psi.vercel.app/adLogin add harish@gmail.com as email and paswword as 123456 to check admin functionality.</li>
</ol>
